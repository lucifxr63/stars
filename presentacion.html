Proyecto de Rediseño

ValidateAI: Bento Dashboard

Transformando reportes estáticos en una experiencia SaaS modular, interactiva y de baja carga cognitiva para fundadores e inversores.

Visión del Producto

El rediseño busca resolver la sobrecarga cognitiva del usuario transformando un reporte estático de texto en un Dashboard SaaS modular e interactivo.

  • Diseño "Dark Mode" premium.
  • Priorización de visualización de datos (Data-Viz).
  • Divulgación progresiva mediante paneles laterales.

Objetivos de Negocio

Aumentar el engagement facilitando la lectura de análisis complejos.

Target Emprendedores, VC, Inversores
MVP Historial Crítico Layout Bento Box responsivo
Side panels para datos crudos
